# Contributing to ephemeris

Thank you for your interest in ephemeris!

## Contribution Policy

This project follows a **closed contribution model**, similar to [SQLite](https://www.sqlite.org/copyright.html). While the source code is freely available under the MIT license, **pull requests are not accepted**.

### Why?

- Maintains a consistent vision and code quality
- Reduces maintenance burden
- Ensures all code has clear provenance

### What You Can Do

- **Use it**: The library is free to use in any project
- **Fork it**: Create your own version with your modifications
- **Report bugs**: Open an issue to report bugs (but fixes will be implemented by the maintainer)
- **Request features**: Open an issue to suggest features (no guarantee of implementation)

### Bug Reports

If you find a bug, please open an issue with:

1. Swift and platform version
2. Minimal code to reproduce
3. Expected vs. actual behavior

### Feature Requests

Feature requests are welcome as issues. Please describe:

1. The use case
2. Why existing functionality doesn't meet your needs

The maintainer will evaluate and implement features as time permits.
